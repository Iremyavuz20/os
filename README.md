The simulator implements and compares the following scheduling algorithms:

- Shortest Job First (SJF)
- Shortest Remaining Time First (SRTF)
- Round Robin (RR)
- Priority Scheduling (Non-Preemptive)
- Priority Scheduling (Preemptive)
